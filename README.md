# 📊 Marketing Campaign Analysis using A/B Testing, Regression Analysis & Time Series Visualization

## 📌 Project Overview

This project analyzes the performance of **Facebook** and **Google AdWords** advertising campaigns using statistical analysis, machine learning, and data visualization techniques.

The objective is to evaluate campaign effectiveness, compare conversion performance, identify trends over time, and provide actionable business recommendations for optimizing advertising strategies.

---

## 🎯 Business Objectives

- Compare Facebook and AdWords campaign performance.
- Analyze customer engagement through clicks and conversions.
- Determine whether Facebook campaigns significantly outperform AdWords.
- Predict Facebook conversions using Linear Regression.
- Analyze monthly and weekly conversion trends.
- Evaluate Cost Per Conversion (CPC) over time.
- Examine the long-term relationship between advertising spend and conversions.

---

---

## 📌 Conclusion

This project demonstrates how statistical analysis and machine learning can be applied to evaluate marketing campaign performance. By combining hypothesis testing, regression analysis, and time-series visualizations, the analysis provides actionable insights into campaign effectiveness, conversion trends, and advertising budget optimization.
